# Hibiscus Event Center Website

A multi-page, responsive marketing website for the Hibiscus Event Center. It is designed to showcase events, venue photos, reviews, and contact information.

## Features
- Responsive layout for desktop and mobile
- Hero section with buttons
- Fixed Navigation Bar with links
- Multiple pages for organized information
- Footer with social links
- Consistency between pages

## Languages Used
- HTML
- CSS
